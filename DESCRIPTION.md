# DashArr Description

DashArr is a unified dashboard for the *arr ecosystem. It consolidates Radarr, Sonarr, Readarr, Prowlarr, and download clients into a single UI and API, providing unified search, queues, and calendar views. The project is built with TypeScript on both the API and web, and is designed for self-hosters who want a clean, fast, all-in-one control surface for their media stack.

## Highlights

- Single dashboard for multiple *arr services
- Unified search, download queue, and calendar
- qBittorrent and SABnzbd integration
- IGDB game discovery and monitoring workflows
- Docker-first deployment options

## Tech Overview

- Backend: Express + TypeScript
- Frontend: React + Vite
- Shared types: TypeScript workspace package
- Infra: Docker and Docker Compose

## Status

Active development, with ongoing integrations and UI improvements.
